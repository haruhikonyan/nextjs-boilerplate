# devcontainer
- あらゆる開発言語に耐えられるよう base image を利用し、言語などは features を使ってインストールする
    - https://mcr.microsoft.com/en-us/artifact/mar/devcontainers/base
- features を使うことで各ライブラリなどのバージョンを細かく指定できる
- OS レベルで必要なライブラリなどが必要な場合は Dockerfile に書き込んでいく
